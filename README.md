GNSS Signal Recording & Replay Tool
A software platform used to record, store, and replay GNSS (GPS, GLONASS, Galileo) signals for testing and simulation. Supports SDR hardware such as AD9361, enabling engineers to validate navigation devices, reproduce real-world GPS scenarios, and debug GNSS-based applications in a controlled environment.
